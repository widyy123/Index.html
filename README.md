# Saya membuat sebuah web mengenai profil saya, dimana penjelasannya:
1. heading
   heading digunakan untuk membuat judul dengan berbagai tipe.
2. br
   br digunakan untuk memberikan jarak antar kalimat
3. hr
   hr digunakan untuk memberikan jarak dan membatasinya dengan garis.
4. ol dan ul
   ol dan juga il digunakan untuk membuat sebuah simbol ataupun angka yang dapat digunakan untuk beberapa pilihan.
5. href
   href digunakan untuk menentukan url atau link yang ditujukkan.
6. td dan juga th
   td digunakan untuk menulis teks tetapi tidak bold, sedangkan th digunakan untuk membuat teks jika bold.
7. border
   border juga digunakan untuk memberikan seberapa banyak ketebalan dari kata tersebut.
8. section
   section digunakan sebagai elemen yang akan mengelompokkan suatu hal.
9. button
   button digunakan untuk membuat sebuah tombol seperti tombol kirim.
10. p
   p digunakan untuk membuat paragraf teks.
